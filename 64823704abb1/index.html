Nginx 安装

Posted on 2022-07-21 In Nginx Views: Disqus: Reading time ≈ 1 mins.

安装

1. 安装依赖环境

  1. 安装 gcc：yum install gcc-c++
  2. 安装 PCRE 库，用于解析正则表达式：yum install -y pcre pcre-devel
  3. zlib 压缩：yum install -y zlib zlib-devel
  4. SSL 安全加密套接字协议层，用于 HTTP 安全传输：yum insta;; -y openssl openssl-devel

2. 安装

  1. 解压得到源码：tar -zxvf nginx-1.17.8.tar.gz

  2. mkdir /var/temp/nginx -p

  3. cd nginx-1.17.8

  4. 创建 makefile 文件

  5. make 编译：make

  6. 安装：make install，完毕之后在 /usr/local/ 下会产⽣⼀个 nginx ⽬录

启动

  • 如果在云服务器安装，需要开启默认的 nginx 端口 80
  • 如果在虚拟机安装，需要关闭防火墙
  • 本地 win 或 mac 需要关闭防火墙
